# -
Mod 简介
本模组添加了一把名为"末影龙鱼竿"的特殊钓鱼竿。用它钓鱼有几率钓上末影龙——不是你想象中的鱼，而是真正的、会攻击你的末影龙。

物品/方块列表
末影龙鱼竿
一把外观和原版钓鱼竿相同的特殊鱼竿
耐久度高达 3000（原版鱼竿仅 64）
附魔能力为 15（与原版鱼竿相同）
无法堆叠（一组只能放 1 把）
合成配方
末影龙鱼竿没有合成配方，目前只能通过创造模式物品栏或指令 /give 获取。

玩法说明
使用方法
手持末影龙鱼竿，对水域右键抛竿（和普通钓鱼一样）
等待鱼漂下沉，再次右键收竿
有 30% 概率 钓上末影龙，而非鱼
钓上末影龙时的效果
末影龙会直接出现在鱼漂的位置
末影龙一出现就会锁定玩家并发动攻击
伴随以下效果：
末影龙怒吼声（声音很大，范围广）
大量紫色传送门粒子
龙息粒子
爆炸粒子
注意事项
末影龙具有攻击性：钓上来后它会立刻追击玩家，请确保自己装备精良、做好准备
消耗耐久度：每次甩竿和收竿都会消耗耐久
生存模式慎用：在主世界召唤末影龙会破坏地形并极大概率导致玩家死亡
不受钓鱼附魔影响：海之眷顾、钓饵等附魔不影响末影龙的触发概率（固定 30%）
概率判定在收竿瞬间：如果触发了末影龙，你不会获得任何鱼
